Show or edit configuration options for github-todos hook in current git repository.
It's essentialy a wrapper around "git config --local", using section "github-todos".

Usage: $0 config [name] [value] [options]

Names:
  inject-issue    Inject issue number in TODO comments (default: false)
  user            Github username (default: extracted from origin)
  repo            Github repository (default: extracted from origin)
